# 🗑️ Garbage Detection System

An AI-powered Computer Vision project that detects garbage and littering in real time using deep learning and OpenCV. Designed for smart surveillance and cleaner urban environments through automated waste detection.

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLO
- NumPy

## ✨ Features
- Real-time garbage detection
- CCTV/video input support
- AI-powered monitoring
- Object detection using deep learning

## 🚀 Future Improvements
- Vehicle number plate recognition
- Automatic fine generation
- Smart city integration
- Real-time alerts
